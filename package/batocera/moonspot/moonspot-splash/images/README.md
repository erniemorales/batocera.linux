# Moon Spot Splash Images

Place your Moon Spot branded images here:

- **logo.png** - Main boot logo (1920x1080 recommended)
- **logo-480p.png** - Lower resolution boot logo for smaller screens (640x480)

These images will be displayed during boot and in the EmulationStation UI.

You can use the provided artwork (the rocket and robot images) as inspiration for your branding.
