# Moon Spot Splash Videos

Place your Moon Spot branded videos here:

- **splash.mp4** - Main boot splash video (720p or 1080p, keep under 10 seconds for fast boot)
- **screensaver/** - Directory for screensaver videos that will play when idle

## Screensaver Videos

The screensaver directory should contain promotional videos, game trailers, or branded content that
will play when the system is idle. Supported formats:
- MP4 (H.264 codec recommended)
- Resolution: 1080p or 720p
- Multiple videos will be played in rotation

Example structure:
```
screensaver/
├── promo1.mp4
├── promo2.mp4
└── gameplay1.mp4
```
